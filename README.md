# Snapperkun

macOS 用のウィンドウスナップツール。ホットキーでアクティブウィンドウの位置・サイズを変更できます。

外部依存なし（システムフレームワークのみ）の Swift 実装です。

## 機能

- アクティブウィンドウを **左右端 / 中央** へ移動
- 幅・高さを **全体 / 3/4 / 2/3 / 1/2 / 1/3 / 1/4** にリサイズ
- 幅・高さ・位置はそれぞれ **現状維持** を選べる（例: サイズを変えずにディスプレイ間移動だけ）
- 「移動 + サイズ」の組み合わせを 1 つのホットキーに割り当て
- 1 つのホットキーに **複数の組み合わせ** を登録し、押すたびに **ローテーション**
- **ディスプレイ間移動**（現在 / 次 / 前のディスプレイを指定して適用）
- メニューバー常駐（Dock アイコンなし。設定ウィンドウ表示中のみ Dock に表示）
- 設定ウィンドウでホットキーとスナップ内容を編集（変更は即保存・即反映）

## 必要環境

- macOS 13.0 以上
- Apple Silicon（現状のバンドルは arm64。Intel は要ユニバーサルビルド）

## ビルドと起動

```sh
# テスト
swift test

# .app バンドルを生成（ad-hoc 署名）
bash Scripts/bundle.sh           # release ビルド
open Snapperkun.app
```

開発中の直接実行は `swift run` でも可能です。

## 使い方

1. 初回起動時に **アクセシビリティ権限** のダイアログが出ます。
   **システム設定 > プライバシーとセキュリティ > アクセシビリティ** で Snapperkun を許可してください
   （他アプリのウィンドウを操作するために必要です）。
2. メニューバーのアイコン（▱）から **「設定…」** を開きます。
3. **初回は設定が空** です。「ホットキー追加」でホットキーを作成します。
   - 追加直後はショートカット **未割り当て** です。「ショートカット」欄をクリックしてキーを入力すると記録されます（`Esc` でキャンセル、✕ ボタンでクリア）。
   - 「サイズ・位置」を追加し、ディスプレイ / 幅 / 高さ / 水平 / 垂直を選びます。
   - サイズ・位置を **複数** 追加すると、そのホットキーを押すたびに循環します。
4. 任意のウィンドウをアクティブにしてホットキーを押すと、設定どおりに移動・リサイズされます。

## 設定ファイル

設定は次の場所に JSON で保存されます。

```
~/Library/Application Support/Snapperkun/settings.json
```
